# Module 1 – Ubuntu Installation & VirtualBox Setup

## Objective
Install Ubuntu 24.04.4 LTS in Oracle VM VirtualBox and configure it for cybersecurity learning.

## Environment
- Ubuntu 24.04.4 LTS
- Oracle VM VirtualBox
- Windows Host

## Topics Covered
- Downloading Ubuntu ISO
- Creating a Virtual Machine
- Configuring RAM, CPU and Storage
- Installing Ubuntu
- Installing VirtualBox Guest Additions
- Enabling Shared Clipboard
- Enabling Drag and Drop
- Updating Ubuntu
- Troubleshooting installation issues

## Skills Acquired
- Ubuntu installation
- Virtual machine configuration
- Linux desktop navigation
- Guest Additions installation
- Basic Linux environment setup

## Outcome
A fully functional Ubuntu virtual machine ready for Linux and cybersecurity training.
